Гарри Поттер на озере

Как-то раз Гарри Поттер вместе со своими друзьями пошёл прогуляться вокруг Хогвартса. Они решили остановиться около озера и поиграть в игру “плюй камни”. Цель данной игры проста: кинуть камень так, чтобы он как можно больше раз отскочил от воды, пока не упадёт в воду. Каждый осткок считается за 1 балл.

Вы, как хороший друг Гарри, записывали результаты каждого участника. Ваша задача назвать 3 людей (или больше при равном числе баллов), кто набрал наибольшее число очков.

Входные данные

На вход дано натуральное число N (3 ≤ N ≤ 106). Далее N строк записаны в следующем формате: имя участника (S - одно слово) и количество набранных им баллом (0 ≤ K ≤ 109).

Выходные данные

Нужно вывести 3-х (или более при равенстве баллов) участников с максимальным количеством баллов в формате: <имя участника> <количество баллов>. Порядок участников с одинаковыми баллами относительно друг друга не важен.
